# FinHelper

一个使用 Python 编写的轻量资产记录应用，支持：

- 按日期记录一次资产快照，默认当天
- 同一次快照录入多个银行/证券账户
- 每个账户同时记录人民币、港元、美元三种金额
- 自动记忆历史账户名称
- 查看最新总资产与历史快照趋势
- 适配电脑端和手机端浏览器

## 启动方式

```bash
pip install -r requirements.txt
python app.py
```

浏览器访问：

```text
http://127.0.0.1:5000
```

## 数据存储

- SQLite 数据库：`assets.db`
- 账户会自动记忆在本地数据库中
- 每次保存都是一条独立资产快照
